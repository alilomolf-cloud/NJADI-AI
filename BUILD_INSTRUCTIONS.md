
# تعليمات بناء تطبيق NJADI AI - ملف الأوامر الكامل

هذا المستند يحتوي على كل ما تحتاجه لبناء التطبيق وتشغيله أو تحويله لملف APK.

## 1. التشغيل المحلي السريع (Quick Run)
افتح الطرفية (Terminal) في مجلد المشروع ونفذ:
```bash
# تثبيت المكتبات
npm install

# تشغيل التطبيق في المتصفح
npm run dev
```

## 2. بناء ملف APK (أندرويد) باستخدام Capacitor
اتبع هذه الأوامر بالترتيب:

### الخطوة أ: التجهيز
```bash
# تثبيت أدوات كاباسيتور
npm install @capacitor/core @capacitor/cli @capacitor/android

# تهيئة المشروع (قم بتغيير المعرف للمعرف الخاص بك)
npx cap init "NJADI AI" com.njadi.ai --web-dir dist
```

### الخطوة ب: البناء والربط
```bash
# بناء نسخة الإنتاج للويب
npm run build

# إضافة منصة الأندرويد
npx cap add android

# مزامنة الملفات (كرر هذا الأمر بعد كل تعديل في الكود)
npx cap sync
```

### الخطوة ج: التوليد (Generation)
```bash
# فتح المشروع في Android Studio
npx cap open android
```
داخل **Android Studio**:
1. انتظر انتهاء الـ Gradle Sync.
2. اذهب إلى القائمة العليا: `Build` -> `Build Bundle(s) / APK(s)` -> `Build APK(s)`.
3. سيظهر تنبيه في الأسفل عند الانتهاء، اضغط على `locate` للحصول على الملف.

## 3. الرفع لـ GitHub واستخدام GitHub Actions
إذا أردت البناء التلقائي:
1. ارفع الكود لـ GitHub.
2. اذهب لـ `Settings` -> `Secrets` -> `Actions`.
3. أضف `GEMINI_API_KEY` (اختياري إذا كنت ستستخدم المفتاح داخل التطبيق).
4. استخدم ملف Workflow لبناء الـ APK تلقائياً عند كل `push`.
